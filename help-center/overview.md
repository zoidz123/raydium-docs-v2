# Support guide

Use this page to understand the right way to contact Raydium support.

If you are a user and need help with the Raydium front end or a product issue, open a ticket in Discord through the support-ticket channel. If you are a developer building on Raydium, use the Devs channel in Discord for integration and builder questions.

## Where to reach out

### User support

Open a Discord support ticket if you:

- ran into a front-end issue
- need help understanding a Raydium product flow
- have a transaction issue and need support

### Developer support

Use the Devs channel in Discord if you:

- are building on Raydium
- have SDK, API, or onchain integration questions
- need help understanding developer docs or implementation details

## How to submit a support ticket

1. Join the official Discord.
2. Go to the support-ticket channel.
3. Click Create ticket.
4. Share the exact product, wallet, transaction signature, screenshots, and error details.

Support replies can take up to 48 hours, though the team aims to respond within that period. Weekend response times may be longer.

## Before you reach out

- Search the docs first to see if your question is already covered.
- Decide whether your question belongs in a support ticket or in the Devs channel.
- Gather the relevant transaction signature, wallet address, screenshots, and exact error details.
- Review [Trust and safety](trust-and-safety.md) before interacting with new links or tokens.

## Important reminders

- Never share your wallet, seed phrase, password, or private key with anyone or any website.
- Always check the full URL of any website you interact with before signing a transaction.
- Always verify the token mint address. Tickers and logos can be spoofed.
- No moderator or support agent should ever ask for direct wallet access.
