# Farms FAQ

## Ecosystem farms

### Can I withdraw my farm rewards after creating a farm?

No. Rewards allocated to farms are final and cannot be withdrawn once the farm is created. Plan emissions carefully before you confirm.

### Why is my farm APR showing 0%?

Rewards only emit when LP tokens are staked in the farm. If nobody has staked yet, APR displays as `0%`.

Stake a small amount of LP tokens yourself to kickstart emissions and display the expected APR.

### Why didn't my farm start exactly on time?

Farm emissions start when your chosen start time reaches Solana cluster time. Depending on network conditions, there may be a slight delay.

### How do I make sure rewards go to the correct pool?

Paste the Pool ID directly instead of searching by token name. Double-check the Pool ID and pool TVL before confirming. Once created, rewards cannot be redirected.

### Can I change the emission rate after creating a farm?

Not during an active period. Wait for the current rewards period to end, then start a new period with an adjusted emission rate.

You can extend the duration at the same rate starting `72` hours before the period ends.

### Can I add more reward tokens later?

Yes. You can add additional reward tokens at any time, up to a maximum of five reward tokens per farm.

### What happens if no one stakes in my farm?

Rewards accumulate but are not distributed until someone stakes. Emissions continue according to schedule and do not pause automatically.

### How long until my farm appears on the UI?

Usually about five minutes after creation. It displays an **Upcoming** tag until the start time.

### Can the same pool have multiple farms?

Yes. The same Pool ID can be used for multiple farms, which allows separate incentive programs on the same pool.
