# How to buy a LaunchLab token

Buy a token on the LaunchLab bonding curve and understand what happens when it graduates.

Use this flow to buy a token while it is still trading on the LaunchLab bonding curve.

> **Info:** Always do your own research and make sure this is the token you want to buy.

## Buy a LaunchLab token

1. Navigate to LaunchLab and search for the token you want to buy.
2. Enter the amount you want to buy.
3. Confirm the transaction. You can sell at any time.

> **Info:** Check your slippage setting before buying or selling.

## What happens after graduation

The token migrates to a Raydium `AMM v4` or `CPMM` pool when the bonding curve is sold through and the graduation target is reached. The default target is `85 SOL`, unless the creator configured a different threshold.

## Important note

LaunchLab is permissionless launch infrastructure for the broader Solana ecosystem. Anyone can create a token and configure bonding-curve parameters, so review the token page carefully before buying.
