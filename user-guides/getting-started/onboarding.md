Set up a Solana wallet, add funds, and get ready to use Raydium.

# Onboarding

To use Raydium, you need a Solana wallet and a small amount of SOL for transaction fees.

- [Wallets](/user-guides/getting-started/quickstart-guides/wallets): Choose a wallet and import or create an account.
- [Buy crypto with fiat](/user-guides/getting-started/quickstart-guides/buy-crypto-with-fiat): Purchase SOL and other assets with MoonPay.
- [Bridge crypto](/user-guides/getting-started/quickstart-guides/bridge-crypto): Move assets from other chains into Solana.

## Before you start

- Keep some SOL in your wallet for network fees.
- Double check token mints before you swap or deposit.
- Use official Raydium links and wallet apps.

After funding your wallet, continue to [the Raydium app](/user-guides/getting-started/the-raydium-app).
