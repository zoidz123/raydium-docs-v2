# AI / LLM

Raydium publishes LLM-optimized text files that give AI coding agents up-to-date documentation context. These are auto-generated from the published docs and stay current as the docs change.

| File          | URL                                                                    | Description                                |
| ------------- | ---------------------------------------------------------------------- | ------------------------------------------ |
| llms.txt      | [docs.raydium.io/llms.txt](https://docs.raydium.io/llms.txt)           | Compact summary with links to each section |
| llms-full.txt | [docs.raydium.io/llms-full.txt](https://docs.raydium.io/llms-full.txt) | Full documentation content inline          |

Use `llms.txt` for quick context. Use `llms-full.txt` if your model has a large context window or you want complete docs without following links.

### Usage

Most AI coding tools can read a URL directly. Point your tool at one of the URLs above.

```
Read the Raydium protocol docs: https://docs.raydium.io/llms.txt
```

```
Read the Raydium protocol docs: https://docs.raydium.io/llms-full.txt
```
