# API Reference

This space is the source of truth for Raydium API references in GitBook.

## Included in this directory

- `specs/openapi.json`
- `specs/trade-api-openapi.json`

## Deploy in GitBook

1. Connect this directory as the `API Reference` space.
2. Keep this README as the landing page.
3. Import `specs/trade-api-openapi.json` into a `Trade API` group.
4. Import `specs/openapi.json` into an `Endpoints` group.
5. Keep generated API pages managed by GitBook rather than hand-authoring endpoint Markdown.

## Base URL

```bash
https://api-v3.raydium.io
```
