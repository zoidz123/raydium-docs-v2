# Main

## Endpoints

- [UI V3 Current Version](version.md): Get the current UI version.
- [UI RPCs](rpcs.md): Get the RPC endpoints used by the UI.
- [Chain Time](chain-time.md): Get the chain time offset relative to local UTC time.
- [TVL and 24-Hour Volume](info.md): Get total value locked and 24-hour volume statistics.
- [RAY Stake Pools](stake-pools.md): Get RAY stake pool data.
- [Migrate LP Pool List](migrate-lp.md): Get LP pools with migration guidance.
- [Transaction Auto Fee](auto-fee.md): Get estimated on-chain fees based on recent transactions.
- [CLMM Config](clmm-config.md): Get the CLMM configs allowed by the UI when creating a pool.
- [CPMM Config](cpmm-config.md): Get the CPMM configs allowed by the UI when creating a pool.
- [Mint Filter Config](mint-filter-config.md): Get the mint filter configs allowed by the UI when creating a pool.
