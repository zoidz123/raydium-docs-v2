# CLMM Position History

## Endpoint

- **Method:** `GET`
- **Path:** `/pools/line/position`
- **Group:** Pools

## Parameters

| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| `id` | query | Yes | `string` | Pool ID. |

## Responses

| Code | Description |
| --- | --- |
| `200` | Successful response. |
