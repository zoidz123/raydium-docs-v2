# Pools

## Endpoints

- [Pool Info](info-ids.md): Get pool info by pool ID.
- [Pool Info by LP Mint](info-lps.md): Get pool info by LP mint.
- [Pool Info List](info-list.md): Get a paginated list of pools.
- [Pool Info by Token Mint](info-mint.md): Get pools by token mint.
- [Pool Info List V2](info-list-v2.md): Get the V2 paginated pool list.
- [Pool Keys](key-ids.md): Get pool keys by pool ID.
- [Pool Liquidity History](line-liquidity.md): Get pool liquidity history for up to 30 days.
- [CLMM Position History](line-position.md): Get CLMM position history for a pool.
