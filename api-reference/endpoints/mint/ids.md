# Mint Info

## Endpoint

- **Method:** `GET`
- **Path:** `/mint/ids`
- **Group:** Mint

## Parameters

| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| `mints` | query | Yes | `string` | Comma-separated IDs. |

## Responses

| Code | Description |
| --- | --- |
| `200` | Successful response. |
