# IDO Pool Keys

## Endpoint

- **Method:** `GET`
- **Path:** `/ido/key/ids`
- **Group:** IDO

## Parameters

| Name | In | Required | Type | Description |
| --- | --- | --- | --- | --- |
| `ids` | query | Yes | `string` | Comma-separated IDs. |

## Responses

| Code | Description |
| --- | --- |
| `200` | Successful response. |
