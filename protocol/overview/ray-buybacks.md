# RAY buybacks

12% of all trading fees across Raydium are used to programmatically buy back RAY.

Buybacks trigger automatically following different logics for each AMM contract. Purchased RAY is held by the protocol.

For on-chain addresses and fund flow details, see [Treasury](/protocol/treasury).
